9.Создание архива с помощью WinRAR и работа с ним
	9.1.Способы запуска программы WinRAR
Запустить WinRAR.exe из папки установки (по умолчанию: C:\Program Files\WinRAR), либо связаный с ним ярлык из любой папки
Запустить WinRAR.exe из окна быстрого запуска, вызываемого комбинацией win+r
Выполнить команду start WinRAR в командной строке

	9.3.Настройка параметров архивации в программе WinRar, самораспаковвывающиеся архивы
Общие параметры архива:
в данной вкладке можно указать имя архива, его  формат, метод сжатия и размер словаря, метод обновления (если архив с таким именем уже существует), и выбрать дополнительные параметры и установить пароль.
Словарь в архивации - условная область поиска в архивируемых файлах, повторения в которой заменяются на образы, которые потом хранятся в одном экземпляре в отдельной таблице. чем больше размер словаря, тем больше компрессия архива


чтобы создать самораспаковывающийся архив в WinRAR, нужно в общих параметрах архива поставить соответсвующую галочку. Вместо простого .rar архива, WinRAR создаст .exe приложение, представляющее собой мастер установки содержимого архива.


	9.4.Для отличников: работа с программой WinRar в командном режиме
Rar.exe  и UnRAR.exe - консольное приложение для работы с архивами в режиме командной строки, находится в папке установки WinRAR (UnRAR - версия для UNIX-систем). Полная справка по  архиватору RAR находится в файле Rar.txt.
Запуск Rar.exe без параметров выводит список возможных параметров с кратким описанием.

чтобы создать архив с одним файлом или добавить файл к существующему архиву нужно написать комманду вида: 
Rar.exe a <имя архива> <путь к файлу>
Чтобы создать архив из множества файлов, сначала нужно создать документ разширения .lst со списком файлов
пример такого файла:

backup.lst:
	c:\work\doc\*.txt      //все текстовые файлы из папки doc 
        c:\work\image\*.bmp    //все .bmp файлы из папки image
        c:\work\misc	       //папка misc со всем её содержимым

тогда комманда создания/расширения архива будет выглядит так: 
Rar.exe a <имя архива> @<путь к списку>

чтобы распаковать архив нужно использавать команду вида:
Rar.exe x <файл архива> <маска извлекаемых файлов(не обязательно)> <каталог выхода(по умолчанию текущий каталог>
