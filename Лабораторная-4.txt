4.Изучение особенностей графического интерфейса. Запуск программ. Использование буфера обмена.
	4.1.Основные элементы графического интерфейса Windows (меню, кнопка Пуск, ярлыки)

	4.2.Основные операции с элементами графического интерфейса Windows (создание, перемещение, переименование, копирование, удаление)

	4.3.Запуск и перемещение по запущенным программам
запустить программы из графического интерфейса можно разными способами:
1.дважды нажать на иконку приложения
2.выбрать ("подсветить") приложение (либо клавишами стрелок, либо одинарным кликом) и нажать enter
 
	4.4.Кнопки управления окнами программ (свернуть, развернуть, закрыть)
кнопка "свернуть" сворачивает окно, убирает его, чтобы оно не отображалось в интерфейсе. Снова открыть окно можно из панели быстрого доступа
кнопка "развернуть" расширяет окно, чтобы оно занимало максимальное кол-во места в оконном режиме. Не путать с полноэкранным режимом.
кнопка "закрыть" закрывает окно, и заканчивает ассоциированные с ним процессы.
 
	4.5.Буфер обмена, основные операции (копирование, вставка)
чтобы скопировать какой-либо тект либо из файла, либо из приложения (если в нём поддерживается выделение текста) используется комбинация клавиш ctrl+c
чтобы вставить в поле ввода скопированный текст, либо любую другую информацию из буфера обмена используется комбинация клавиш ctrl+v

	4.6.Для отличников: буфер обмена в командной строке, доступ к папке расположения ярлыков и папок Рабочего стола


чтобы вставить текст в командную строку надо кликнуть ПКМ на окно терминала и выбрать там "вставить" (или "paste")

содержимое рабочего стола находится в соответствующей папке в каталоге пользавателя
